# Barbería Arte en Tijeras – Sitio Web

Descripción del Proyecto

Este proyecto consiste en la creación de un sitio web completo para la barbería Arte en Tijeras. El objetivo es aplicar los conocimientos adquiridos durante el curso utilizando HTML, CSS y JavaScript para desarrollar un sitio funcional, organizado y visualmente atractivo.

Contenido del Sitio

El sitio está compuesto por varias páginas que representan diferentes áreas de la barbería:

Home: Página principal de bienvenida.

About: Información sobre la barbería y su propósito.

Shop: Productos con un carrito de compra básico.

Appointment: Formulario para reservar citas.

Membership: Planes y beneficios para clientes frecuentes.

Gallery: Galería de imágenes con filtro y visualizador ampliado.

Contact: Formulario de contacto con validación.

Tecnologías Utilizadas

HTML5

CSS3

JavaScript

Estas tecnologías se usan para estructurar, diseñar e implementar interactividad dentro del sitio.

Funcionalidades Interactivas

Confirmación emergente al reservar una cita.

Carrito de compras con almacenamiento local.

Validación del formulario de contacto.

Galería interactiva con lightbox.

Cómo Ejecutar el Proyecto

Descarga o clona el repositorio.

Abre la carpeta del proyecto en Visual Studio Code.

Inicia el archivo home.html con Live Server o ábrelo directamente desde tu navegador.
